User Experience Design Projects

This is a collection of the User Experience and User Interface design projects I have worked on.

Online grocery store interface redesign

I remotely collaborated on a team of four people to redesign the interface for online grocery shopping. We extensively used user interviews and user co-design sessions to assure that experiences users valued in a traditional grocery store settings were being accounted for and that the experience was intuitive and natural for users to use.

I also used HTML and CSS to develop and maintain a website which we use to showcase the work we performed on the project, as well as a functional Figma prototype of our online store interface.

Learn More and Visit the Website

Website Showcase For an autonomous turret

This is a website I developed in collaboration with three teammates to showcase the work we we performed for an autonomous sentry turret which we developed. We used HTML and CSS to program the back end components of the website. Myself and one teammate maintain the site and keep it functional

Sound Labyrinth Website

I work with one other person to develop and maintain this website with Html and CSS to showcase the work i did on an accessible video game. The game is a dungeon crawler video game which is designed to provide the same User Experience through both visual and Audio queues. Thus it provides the same experience to users regardless of their visual or audio ability.
